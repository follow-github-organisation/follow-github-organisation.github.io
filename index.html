How it works

How to follow an organization?

Go to an organization page, click on the "Follow" button in the top right if you want to see the new activities by that organization in your feed.

Where will the activity be shown?

Whenever a new repository is created in your followed organizations. The activity will be added to your main GitHub feed.

FAQ

  • What is Follow-Github-Organization?

    It's a browser extension that lets you follow a GitHub organization like you would follow a normal user account. It shows you new activities (repository creation as of now) in the organizations in your GitHub feed.

  • What if GitHub releases this as a feature?

    That's the point. Win-win situation. Mahatma Gandhi once said, "Create the pull request you want to see in the repository".

  • How much does it cost?

    There are two versions of it, "Adopter" and "Supporter". The former version is free of cost, the later costs $15.

  • What is the difference between Adopter and Supporter versions?

    The supporter version allows you to follow as many organizations as you want. Wherein the adopter version lets you follow only 10 organizations.
